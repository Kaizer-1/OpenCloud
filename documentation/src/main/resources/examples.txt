Running the OpenCloud examples
-----------------------------

To run the OpenCloud examples you need to do the following steps.

In Windows:

1. cd <PATH TO opencloud PACKAGE>\jars
2. java -classpath opencloud-<VERSION>.jar;opencloud-examples-<VERSION>.jar org.opencloud.opencloud.examples.opencloudExample<EXAMPLE NUMBER>

In Unix/Linux:

1. cd <PATH TO opencloud PACKAGE>/jars
2. java -classpath opencloud-<VERSION>.jar:opencloud-examples-<VERSION>.jar org.opencloud.opencloud.examples.opencloudExample<EXAMPLE NUMBER>

Where you need to replace:

<PATH TO opencloud PACKAGE> - by the path to a directory where you have unpacked the OpenCloud package
<VERSION> - by the version of the downloaded OpenCloud package
<EXAMPLE NUMBER> - by the of number of the example you want to run


OpenCloud examples source code
-----------------------------

You can find the source code of the examples in <PATH TO opencloud PACKAGE>/examples/org/opencloud/opencloud/examples/


Compiling and running examples
------------------------------

To compile and run an example (let's say org.opencloud.opencloud.examples.opencloudExample1) you need to do the following steps:

In Windows:

1. cd <PATH TO opencloud PACKAGE>
2. javac -classpath jars\opencloud-<VERSION>.jar;examples examples\org\opencloud\opencloud\examples\opencloudExample1.java
3. java -classpath jars\opencloud-<VERSION>.jar;examples org.opencloud.opencloud.examples.opencloudExample1

In Unix/Linux:

1. cd <PATH TO opencloud PACKAGE>
2. javac -classpath jars/opencloud-<VERSION>.jar:examples examples/org/opencloud/opencloud/examples/opencloudExample1.java
3. java -classpath jars/opencloud-<VERSION>.jar:examples org.opencloud.opencloud.examples.opencloudExample1


Compiling and running examples from the power package
-----------------------------------------------------

To compile and run an example (let's say org.opencloud.opencloud.examples.power.planetlab.LrMc) you need to do the following steps:

In Windows:

1. Download Michael Thomas Flanagan's Java Scientific Library from http://www.ee.ucl.ac.uk/~mflanaga/java/
2. Copy flanagan.jar to <PATH TO opencloud PACKAGE>/jars/
3. cd <PATH TO opencloud PACKAGE>
4. javac -classpath jars\opencloud-<VERSION>.jar;jars\flanagan.jar;examples examples\org\opencloud\opencloud\examples\power\planetlab\LrMc.java
5. java -classpath jars\opencloud-<VERSION>.jar;jars\flanagan.jar;examples org.opencloud.opencloud.examples.power.planetlab.LrMc

In Unix/Linux:

1. cd <PATH TO opencloud PACKAGE>
2. chmod +x ./install-flanagan.sh
3. ./install-flanagan.jar
4. javac -classpath jars/opencloud-<VERSION>.jar:jars/flanagan.jar:examples examples/org/opencloud/opencloud/examples/power/planetlab/LrMc.java
5. java -classpath jars/opencloud-<VERSION>.jar:jars/flanagan.jar:examples org.opencloud.opencloud.examples.power.planetlab.LrMc


Description of the OpenCloud examples
------------------------------------

Here is the description of what each example does:

opencloudExample1.java : shows how to create a datacenter with one host and run one cloudlet on it.

opencloudExample2.java : shows how to create a datacenter with one host and run two cloudlets on it.
			The cloudlets run in VMs with the same MIPS requirements. The cloudlets will take the same 
			time to complete the execution.

opencloudExample3.java : shows how to create a datacenter with two hosts and run two cloudlets on it.
			The cloudlets run in VMs with different MIPS requirements. The cloudlets will take different 
			time to complete the execution depending on the requested VM performance.

opencloudExample4.java : shows how to create two datacenters with one host each and run two cloudlets on them.

opencloudExample5.java : shows how to create two datacenters with one host each and run cloudlets of two users on them. 

opencloudExample6.java : shows how to create scalable simulations.

opencloudExample7.java : shows how to pause simulations.

opencloudExample8.java : shows how to add entities in run time.

network: this package contains examples on how to run simulation with network simulation.

power: this package contains examples on how to use OpenCloud's power-aware features.
